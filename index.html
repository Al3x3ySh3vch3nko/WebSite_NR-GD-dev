[Этап 3] Cогласительные процедуры 

// При получении замечаний ФОИВ:
- разработка Протокола согласительного совещания и Таблицы разногласий \ учета замечаний (в 2х экземплярах);
- согласование в рабочем порядке с исполнителем текстов Протокола \ Таблицы
- согласование в ОСЭД (раздел - ОРД) с курирующим замМинистром (СП) Протокола \ Таблицы;
- завизировать у курирующего замминистра согласованный в электронном виде Протокол \ Таблицу (в 2х экземплярах);
- направление за подписью курирующего замминистра завизированных оригиналов Протокола \ Таблицы (в 2х экземплярах);
- получение подписанного экземпляра Протокола \ Таблицы;
- загрузка в ОСЭД в раздел ОРД скана с подписями для регистрации;
- подписывается курирующим замМинистром и регистрируется приемной.

>> Срок:
- согласование таблицы разногласий: 3 дня.
- направление и согласование с ФОИВ: 15 дней.
Всего: 3-20 дней.
